Enhancement 1: Fixed Navigation Bar

  • Fixed Navigation Bar is the one that maintains its position on the screen while allowing user to see different sections of a website without having to scroll the page.
    Fixed navigation bar allows webpage to be more responsive to the user interaction.
    To create fixed navigation bar use 'fixed' element for 'position' value.

  • Code:

    CSS File:

                .nav-bar {
                overflow: hidden;
                background-color:white;
                position: fixed;
                width: 100%;
                margin-bottom: 2%;
                top: 0%;
                margin-top: 0%;
              }

  • Link to the extension: Fixed Navigation Bar
  • Sources Reference: W3schools horizontal navigation bar

Enhancement 2: Button Pressed Effect

  • Button Pressed Effect allow the user to experience interaction with the button on the webpage.
    Specifically this effect allow the button when clicked by the user to be move down just like buttons in real world.
    To create a button pressed down effect, we combine the 'Translate()'' method to the active part of the button in CSS along with the 'box-shadow' property.
    TranslateX() and TranslateY() method in CSS allow an element to move position with distance set before.
    The 'box-shadow' porperty adds shadow to the element.

  • Code:

    CSS File:

                        #formApplyButton:active {
                        background:rgb(82, 137, 82);
                        
                        box-shadow: 0 5px #777;
                        transform: translateY(4px);      
                        /*Move button down 4px when click it*/
                      }
                      
                      #formResetButton:active {
                        background:rgb(142, 76, 76);
                      
                        box-shadow: 0 5px #777;
                        transform: translateY(4px);
                        /*Move button down 4px when click it*/
                      }

  • Link to the extension: Button Transformation
  • Sources Reference: W3s Animated Button
